# The Layered Light

A gift page and printable card for a handmade sacred-geometry necklace — three
layered pendants: the **Flower of Life** (polished steel), **Metatron's Cube**
(polished steel), and the **Tree of Life** (solid gold), each form hidden inside
the last.

**Live page:** https://zev330-lab.github.io/necklace/

## Contents
- `index.html` — the gift page. Single self-contained file (photos embedded, SVG
  diagrams generated in-page, QR generated at runtime from the live URL). Dark,
  mystical theme; mobile-first.
- `card.pdf` — 5×7in printable card, 300 DPI, 0.125in bleed + crop marks.
- `card.jpg` — trimmed card for texting/sharing.

The page covers the geometry of each form, its place in Kabbalah and in the wider
"sacred geometry" tradition, what each is worn to carry, and how to hold the three
in mind. The card's QR code links to this page.
